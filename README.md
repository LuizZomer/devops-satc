## 📌 DevOps

DevOps é uma **cultura e conjunto de práticas** que integra Desenvolvimento (Dev) e Operações (Ops) para acelerar a entrega de software com qualidade.

### Objetivos principais:
- Automatizar processos
- Melhorar colaboração entre equipes
- Aumentar frequência de deploy
- Reduzir falhas

---

## 📌 SRE (Site Reliability Engineering)

SRE é uma abordagem criada pelo Google que aplica engenharia de software para garantir **confiabilidade e estabilidade** dos sistemas.

### Foco principal:
- Alta disponibilidade
- Monitoramento com métricas (SLI, SLO, SLA)
- Controle de erro (Error Budget)
- Automação operacional
